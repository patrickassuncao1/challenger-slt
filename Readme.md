# Desafio Slt

## Iniciar a aplicação

- docker compose  up -d 
- Entra no container php e rodar o comando composer install

- Aplicação em http://localhost:8080

## Conexão com o banco
- criar um arquivo .env e modificar as seguintes chaves com as credecias do seu banco
```
DB_HOST=mysql
DB_PORT=3306
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=

```

- scripts sql em ./script.sql


## Versão PHP 7.4
